Super-search query targeting various record types
    ==================================================
    A) Record sets for each record type are retrieved matching query 
       for each record type.
    
        - Each record, no matter what record type, has an associated 
          protein identifier. 
    
        - For protein/enzyme/glycoprotein record types, the UniProtKB cannonical accession 
          is the protein identifier. 
    
        - For site record types, the protein identifier is the UniProtKB 
          cannonical accession of the protein in which the site is located.
    
        - For glycan record types, the protein  identifier is the UniProtKB 
          cannonical accession of the glycoprotein glycosylated by the glycan. 
   

    B) A record from each set found in (A) is removed if its associated
       protein identifier is not found in all other sets. This is done to 
       ensure maintain the relationship in the returned records.



    {
        "glycan": {
            "$and": [
                {"species.common_name": {"$eq": "Human"}},
                {"motifs.name": {"$regex": "Polysialic", "$options": "i"}},
                {"mass": {"$gte": 2100}}
            ]
        }, 
        "protein": {
            "$and":[
                {"expression_tissue.tissue.name":{"$regex": "lung", "$options": "i"}},
                {"expression_tissue.present":{"$regex": "yes", "$options": "i"}},
                {"expression_disease.significant":{"$regex": "yes", "$options": "i"}},
                {"expression_disease.disease.recommended_name.name":{"$regex": "lung", "$options": "i"}}
            ]
        }, 
        "site": {
            "$and":[
                {"snv":{"$gt":[]}},
                {"snv.disease":{"$gt":[]}},
                {"glycosylation":{"$gt":[]}},
                {"glycosylation.glytoucan_ac":{"$ne":""}},
                {
                    "$or":[
                        {"species.taxid":{"$eq":9606}},
                        {"species.name":{"$regex": "homo", "$options": "i"}},
                        {"species.common_name":{"$regex": "human", "$options": "i"}}
                    ]
                }
            ]
        }
    }
